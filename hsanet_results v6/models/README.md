# Model checkpoints

Trained `.pth` files are **not** stored in this Git repository due to size (40–512 MB per file).

To reproduce results, train using `hsanet-v6.ipynb` on [BRISC 2025 (Kaggle)](https://www.kaggle.com/datasets/briscdataset/brisc2025).

For a copy of our trained weights, contact **assaduzzaman.cse@diu.edu.bd** or open an issue on https://github.com/tarequejosh/HSANet.
